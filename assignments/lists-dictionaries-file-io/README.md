# 📘 Assignment: Lists, Dictionaries, and File I/O

## 🎯 Objective

Practice organizing and saving student data with Python lists, dictionaries, and text file input/output. Students will build a small program that stores information, updates it, and reads it back from a file.

## 📝 Tasks

### 🛠️ Build a Student Record List

#### Description
Create a program that stores several student records using a list of dictionaries.

#### Requirements
Completed program should:

- Define a list of student dictionaries with fields such as name, grade, and favorite subject
- Display all students in a clear, readable format
- Add at least one function to find or summarize the students

### 🛠️ Save and Load Data from a File

#### Description
Save the student records to a text file and then load them back into the program.

#### Requirements
Completed program should:

- Write the student records to a file named `students.txt`
- Read the saved data back into the program
- Show that the loaded information matches the original records

### 🛠️ Add a Simple Update Feature

#### Description
Allow the user to add a new student record and update the saved list.

#### Requirements
Completed program should:

- Prompt for a new student name, grade, and favorite subject
- Add the new record to the existing list of students
- Save the updated list back to the file
